## git 사용법
------------
  
```
terminal에 입력  
  
git config --global user.name " "
git config --global user.email " "  
  
echo "# 레포지토리명" >> READ.md 리드미파일을 생성(프로그램 설명서)
git init (git 초기화)
git add README.md 
git commit -m " "
git branch -M main
git remote add origin http://git주소
git push -u origin main
```
